# Front (Legado)

Este diretório representa uma versão antiga/alternativa do front-end.
Para evitar duplicidade de deploy e confusão com o App Router principal, **não** faça deploy a partir de `Front/`.

- App principal: raiz do projeto (`app/`, `components/`, `lib/`)
- Deploy no Railway deve apontar para a raiz

Se desejar arquivar, mova `Front/` para `archive/Front/` quando o editor não estiver utilizando arquivos daqui.
